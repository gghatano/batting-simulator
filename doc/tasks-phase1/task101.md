# Task 101: 選手カードのコンパクト化（CSS Grid + 表示項目の削減）

## フェーズ

Phase 1 - UI改善

## 依存タスク

なし

## 内容

選手カードのサイズを縮小し、一覧の表示密度を上げる。一覧を CSS Grid で実装し、レスポンシブに列数を可変にする。

### やること

* `PlayerList.svelte` のレイアウトを CSS Grid に変更
  * `grid-template-columns: repeat(auto-fill, minmax(180px, 1fr))`
* `PlayerCard.svelte` をコンパクト版に変更
  * 表示項目: 名前、チーム、PA、HR/PA、BB+HBP/PA、SO/PA、OPS（任意）
  * 非表示にする項目: 1B/2B/3B 等の内訳（詳細表示へ移動、Task 102 で実装）
* カード自体のパディング・フォントサイズを縮小

## 完了条件

- [ ] 選手一覧が CSS Grid で表示される
- [ ] 画面幅に応じて列数が自動変化する（レスポンシブ）
- [ ] カード表示が名前、チーム、PA、HR/PA、BB+HBP/PA、SO/PA に絞られている
- [ ] 以前より多くの選手カードが一画面に表示される
- [ ] `npm run build` が成功する

## テスト

* 手動確認: ブラウザ幅を変更して列数が変化すること
* 手動確認: カード内の表示項目が削減されていること
