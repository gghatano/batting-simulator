# Task 102: 選手カードの詳細表示（モーダル）

## フェーズ

Phase 1 - UI改善

## 依存タスク

* Task 101（コンパクト化後のカードに「詳細」ボタンを追加するため）

## 内容

コンパクト化されたカードに「詳細」ボタンを追加し、押下で詳細情報をモーダル表示する。

### やること

* `src/components/PlayerDetailsModal.svelte` を新規作成
* `PlayerCard.svelte` に「詳細」ボタンを追加
* モーダルに以下を表示:
  * 選手名、チーム
  * 1B/2B/3B/HR の率と生値
  * K率、BB+HBP率
  * OUT率（計算値）
  * PA
* モーダルの閉じ方: ×ボタン、背景クリック、Escape キー

## 完了条件

- [ ] カードに「詳細」ボタンが存在する
- [ ] 「詳細」ボタン押下でモーダルが表示される
- [ ] モーダルに 1B/2B/3B/HR の率と生値、K率、BB+HBP率、OUT率が表示される
- [ ] ×ボタン、背景クリック、Escape でモーダルが閉じる
- [ ] `npm run build` が成功する

## テスト

* 手動確認: 詳細ボタン → モーダル表示 → 各情報が正しい
* 手動確認: 3つの閉じ方がそれぞれ動作する
