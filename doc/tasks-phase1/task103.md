# Task 103: 打線パネルに「シミュレーション開始」ボタン追加

## フェーズ

Phase 1 - UI改善

## 依存タスク

なし

## 内容

画面1（打線作成）の右側パネルに「シミュレーション開始」ボタンを追加し、打線完成時にシミュレーション画面へ遷移できるようにする。

### やること

* `LineupPanel.svelte` の「全クリア」ボタンの下に「シミュレーション開始」ボタンを追加
* 打線（9枠）が未完成の場合:
  * ボタン押下でエラーメッセージを表示（例:「9人全員を設定してください」）
  * 遷移しない
* 打線が完成している場合:
  * `activeTab` を `'simulation'` に切り替えてシミュレーション画面へ遷移

### やること（補足）

* `src/stores/ui.ts` の `activeTab` を import して使用

## 完了条件

- [ ] 「シミュレーション開始」ボタンが打線パネルに存在する
- [ ] 打線完成時にボタン押下でシミュレーション画面へ遷移する
- [ ] 打線未完成時にボタン押下でエラーメッセージが表示される
- [ ] 打線未完成時にシミュレーション画面へ遷移しない
- [ ] `npm run build` が成功する

## テスト

* Vitest: 打線の完成判定ロジック（9枠すべて非null → true, 1つでもnull → false）
* 手動確認: 未完成打線でボタン押下 → エラー表示、遷移しない
* 手動確認: 完成打線でボタン押下 → シミュレーション画面へ遷移
