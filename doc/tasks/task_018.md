# Task 018: `StatsTable` 打者別率・打線合計結果

## フェーズ

5. Worker化と画面2

## 依存タスク

* Task 012
* Task 017

## 内容

打線の各打者のイベント率と、シミュレーション結果の統計量をテーブル表示するコンポーネントを実装する。

### やること

* `src/components/StatsTable.svelte` を作成
* 打者別イベント率テーブル:
  * 行: 1番〜9番の各打者
  * 列: 打順、選手名、p1B、p2B、p3B、pHR、pBB_HBP、pK、pOUT
  * 率は小数3桁表示
* シミュレーション結果サマリ:
  * 平均得点、中央値、P10、P90
  * SimulationPanel 内または隣接して配置

## 完了条件

- [ ] 打者別の全イベント率がテーブルで表示される
- [ ] 各率が小数3桁で表示される
- [ ] 全率の合計が各行で 1.000（表示上）
- [ ] シミュレーション結果（mean/median/p10/p90）が表示される

## テスト

* 手動確認: 打線を組んで画面2に遷移 → 打者別率テーブルが表示される
* 手動確認: シミュレーション実行後に統計量が更新される
* 手動確認: 率の合計が各打者で 1.000 前後
