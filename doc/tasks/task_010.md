# Task 010: PlayerCardの「追加」→選択枠にセット

## フェーズ

3. 打線作成UI

## 依存タスク

* Task 007
* Task 009

## 内容

PlayerCard の「追加」ボタンを押すと、現在選択中の打線枠にその選手がセットされるようにする。

### やること

* `PlayerCard.svelte` の「追加」ボタンに click ハンドラを追加
  * `selectedSlotStore` の値を取得し、`setSlot(selectedSlot, player)` を呼ぶ
  * セット後、`selectedSlotStore` を次の空き枠（または次の番号）に自動移動させる
* 画面1のレイアウト統合:
  * 左カラム: `PlayerList`
  * 右カラム: `LineupPanel`

## 完了条件

- [ ] 「追加」ボタン押下で選択中の枠に選手がセットされる
- [ ] セット後に選択枠が次に移動する
- [ ] 左右カラムレイアウトで打線作成画面が機能する
- [ ] 同一選手を複数枠に追加できる

## テスト

* 手動確認: 枠1を選択 → カードの「追加」→ 枠1に選手名が表示
* 手動確認: 続けて「追加」→ 枠2に別の選手が入る
* 手動確認: 同じ選手を2回追加 → 2枠に同一選手が入る
