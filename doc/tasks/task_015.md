# Task 015: `rng.ts` seed対応RNG

## フェーズ

4. 率計算とシミュレーション（ロジック）

## 依存タスク

なし

## 内容

シミュレーションの再現性を担保するため、seed 指定可能な疑似乱数生成器を実装する。

### やること

* `src/lib/sim/rng.ts` を作成
* `createRng(seed: number): () => number`
  * seed から初期化された PRNG を返す
  * 戻り値の関数: 呼ぶたびに `[0, 1)` の疑似乱数を返す
  * アルゴリズム: mulberry32 等の軽量PRNG
* seed 未指定時は `Math.random()` ベースのフォールバック（またはランダム seed 生成）

## 完了条件

- [ ] 同じ seed で生成した RNG が同じ数列を返す
- [ ] 異なる seed で異なる数列が返る
- [ ] 戻り値が `[0, 1)` の範囲に収まる
- [ ] 十分な周期を持つ（短い周期で繰り返さない）

## テスト

* Vitest:
  * 同一 seed → 100回呼び出した結果が完全一致
  * 異なる seed → 結果が異なる
  * 10000回呼び出して全て `0 <= x < 1` の範囲内
  * 分布の偏りチェック（簡易: 0.5未満と0.5以上がおおよそ半々）
