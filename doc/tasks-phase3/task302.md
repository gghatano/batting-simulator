# Task 302: ポジション情報の追加とカード色分け

## フェーズ

Phase 3 - UX改善・機能追加

## 依存タスク

* Task 301（カードのレイアウト変更後に適用）

## 背景

パワプロ（実況パワフルプロ野球）を参考に、選手カードをポジション別に色分けし、ポジションを1文字で表示したい。現状の `players.json` にはポジション情報がないため、データモデルの拡張から行う。

## 内容

### やること

* **データモデル拡張**:
  * `Player` インターフェースに `position` フィールドを追加（string型）
  * 値: `"投"`, `"捕"`, `"一"`, `"二"`, `"三"`, `"遊"`, `"左"`, `"中"`, `"右"`, `"指"` （パワプロ準拠の1文字表記）
* **`public/players.json` の更新**:
  * 全選手に `position` フィールドを追加
  * 架空データなので適切に割り振る（チームごとにバランス良く）
* **ポジション別カード色（パワプロ準拠）**:
  * 投手（投）: ピンク（`#fce4ec` / `#e91e63` 系）
  * 捕手（捕）: 水色（`#e0f2f1` / `#00bcd4` 系）
  * 内野（一・二・三・遊）: ライトイエロー（`#fff9c4` / `#ffc107` 系）
  * 外野（左・中・右）: 薄緑（`#e8f5e9` / `#4caf50` 系）
  * 指名打者（指）: ライトグレー（`#f5f5f5` / `#9e9e9e` 系）
* **`PlayerCard.svelte` の変更**:
  * カード背景色をポジションに応じて変更（左ボーダーまたは背景色）
  * ポジション1文字をカード上に表示（バッジ形式: 小さな丸 or 角丸チップ）
* **`src/styles/tokens.css` にポジションカラーを定義**:
  * `--color-pos-pitcher`, `--color-pos-catcher`, `--color-pos-infield`, `--color-pos-outfield`, `--color-pos-dh`
  * それぞれ背景用（薄い色）とアクセント用（濃い色）のペア
* **フィルター対応**:
  * `PlayerList.svelte` のフィルターにポジション絞り込みを追加（任意: セレクトボックスまたはチップ）
