# Task 303: 打線パネルの幅固定と成績表示

## フェーズ

Phase 3 - UX改善・機能追加

## 依存タスク

* Task 302（ポジション情報追加後）

## 背景

打線パネルの幅が、名前の有無やタブ切替で動的に変わってしまう。打線作成とシミュレーション画面で同じ枠・同じ位置に固定表示したい。また、打線スロットにはチーム名よりも選手成績を表示してほしい。

## 内容

### やること

* **打線パネルの幅固定**:
  * `LineupPanel.svelte`: `width` を固定値に設定（例: `320px` または `22rem`）
  * `LineupSummary.svelte`: 同じ幅に固定
  * 打線作成タブ・シミュレーションタブの両方で同じ位置に表示されるように
  * `App.svelte` の `.layout` / `.sim-layout` で打線パネルの位置を統一
* **打線スロットの成績表示**:
  * `LineupSlot.svelte`: チーム名（`slot.team`）の代わりに成績を表示
  * 表示内容: 打率（AVG）と HR率 の2つ
    * 打率 = `(single + double + triple + hr) / pa`
    * HR率 = `hr / pa`
  * フォーマット: `.285 / HR.033` のような形式
  * 小さめのフォント（`--font-xs`）、muted カラー
* **ポジション表示**（Task 302 完了後）:
  * スロットにポジション1文字バッジを追加（打順番号バッジの横、または別の位置）

## 完了条件

- [ ] 打線パネルの幅がタブ切替で変わらない
- [ ] 打線スロットに打率・HR率が表示されている

- [ ] 打線作成とシミュレーションで打線パネルの位置が同じ
