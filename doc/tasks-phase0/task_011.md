# Task 011: 枠クリア／全クリア／上下入れ替え

## フェーズ

3. 打線作成UI

## 依存タスク

* Task 009
* Task 010

## 内容

打線パネルの枠操作UIを実装する。

### やること

* `LineupSlot.svelte` に以下のボタンを追加:
  * 「×」（枠クリア）: `clearSlot(index)` を呼ぶ
  * 「↑」（上に移動）: `swapSlots(index, index - 1)`。1番枠では非表示 or disabled
  * 「↓」（下に移動）: `swapSlots(index, index + 1)`。9番枠では非表示 or disabled
* `LineupPanel.svelte` に「全クリア」ボタン: `clearAll()` を呼ぶ
* 空枠にはクリア・入れ替えボタンを表示しない（または disabled）

## 完了条件

- [ ] 各枠の「×」ボタンでその枠のみクリアされる
- [ ] 「↑」「↓」ボタンで隣接枠と入れ替わる
- [ ] 1番枠の「↑」と9番枠の「↓」が無効化されている
- [ ] 「全クリア」で全枠が空になる
- [ ] 空枠では操作ボタンが非活性

## テスト

* 手動確認: 選手を数枠にセット → 「×」で個別クリア → 該当枠のみ空になる
* 手動確認: 1番と2番に選手セット → 1番の「↓」→ 2選手が入れ替わる
* 手動確認: 「全クリア」→ 全枠が空になる
* 手動確認: 1番枠の「↑」ボタンが disabled / 9番枠の「↓」が disabled
