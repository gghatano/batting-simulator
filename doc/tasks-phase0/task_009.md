# Task 009: `LineupPanel`（1〜9枠）と投入先選択

## フェーズ

3. 打線作成UI

## 依存タスク

* Task 008

## 内容

打線パネルUIを実装する。9つの打順枠を表示し、投入先の枠を選択できるようにする。

### やること

* `src/components/LineupSlot.svelte`
  * 打順番号（1〜9）を表示
  * 選手がセットされている場合: 選手名と主要スタッツを表示
  * 空の場合: 「未設定」等のプレースホルダ表示
  * クリックで `selectedSlotStore` を更新（投入先選択）
  * 選択中の枠にはハイライト表示
* `src/components/LineupPanel.svelte`
  * 9つの `LineupSlot` を縦に並べて表示
  * 「全クリア」ボタン

## 完了条件

- [ ] 9枠が1番〜9番として表示される
- [ ] 枠クリックで選択状態が切り替わる（ハイライト表示）
- [ ] 選手セット済みの枠に選手名が表示される
- [ ] 空枠に「未設定」等が表示される

## テスト
