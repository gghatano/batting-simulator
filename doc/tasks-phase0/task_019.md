# Task 019: 得点分布の表示

## フェーズ

5. Worker化と画面2

## 依存タスク

* Task 017

## 内容

シミュレーション結果の得点分布を表形式で表示する。（チャートは後回し）

### やること

* `SimulationPanel.svelte` または専用コンポーネントに得点分布テーブルを追加
* テーブル構成:
  * 行: 0点、1点、2点、…（上限は動的: 最大得点まで）
  * 列: 得点、出現回数、出現率（%）
* `simResultStore` の `distribution`（`Map<number, number>`）を使用
* 得点の昇順でソート

## 完了条件

- [ ] シミュレーション実行後に得点分布テーブルが表示される
- [ ] 0点から最大得点まで全行が表示される
- [ ] 出現回数と出現率（%）が表示される
- [ ] 出現回数の合計が N と一致する
- [ ] 出現率の合計がおおよそ 100%

## テスト

* 手動確認: N=10000 で実行 → 分布テーブルの出現回数合計が 10000
* 手動確認: 全打者 pOUT=1.0 → 0点が 100%
* 手動確認: 得点が昇順で並んでいる
