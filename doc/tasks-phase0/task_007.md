# Task 007: `PlayerCard` と `PlayerList` 実装

## フェーズ

2. データ読み込みと一覧

## 依存タスク

* Task 006

## 内容

選手カードコンポーネントとカード一覧コンポーネントを実装する。

### やること

* `src/components/PlayerCard.svelte`
  * 表示項目: 名前、チーム、PA、1B/2B/3B/HR、BB+HBP、SO
  * 派生率表示: 各イベント/PA（小数3桁）
  * 「追加」ボタン（打線への追加用、Task 010 で接続）
* `src/components/PlayerList.svelte`
  * `filteredPlayersStore` を subscribe して一覧表示
  * チーム選択ドロップダウン
  * 名前検索入力欄
  * ソート選択ドロップダウン

## 完了条件

- [ ] 選手カードに名前・チーム・各スタッツ・派生率が表示される
- [ ] 派生率が小数3桁で表示される（例: `.285`）
- [ ] PlayerList にフィルタ・検索・ソートのUI操作が存在する
- [ ] フィルタ変更時にカード一覧がリアクティブに更新される
- [ ] カードに「追加」ボタンが存在する（この時点では動作不要）

## テスト

* 手動確認: ブラウザで選手カードが一覧表示される
* 手動確認: チーム切替・名前検索・ソート変更で一覧が更新される
* 手動確認: 派生率の計算結果が妥当（PA=500, HR=30 → HR率 .060）
