# Task 205: ヘッダー・ナビゲーションの改善

## フェーズ

Phase 2 - UIデザインブラッシュアップ

## 依存タスク

* Task 204（テーマカラー確定後に適用）

## 内容

アプリヘッダーとタブナビゲーションを刷新し、アプリらしい見た目にする。

### やること

* `App.svelte` のヘッダー部分をリデザイン:
  * ヘッダー領域を作成: テーマカラー背景 + 白テキスト
  * アプリ名「Batting Simulator」をヘッダー内に配置
  * アイコン追加（⚾ や SVG）は任意
* タブナビゲーションをヘッダー下に統合:
  * タブを大きくする（padding 拡大、font-size 拡大）
  * アクティブタブに下線インジケータ（3px テーマカラー）
  * アクティブタブの背景色を明確に
  * タブにラベルアイコンを追加（任意: ⚾ 打線作成、📊 シミュレーション）
* ヘッダーを `position: sticky; top: 0` で固定（任意）

## 完了条件

- [ ] テーマカラー背景のアプリヘッダーが存在する
- [ ] タブが視覚的に明確で、アクティブ状態がはっきり分かる
- [ ] ヘッダー + タブが一体的なナビゲーションに見える
- [ ] `npm run build` が成功する

## テスト

* 手動確認: ヘッダーがテーマカラーで表示される
* 手動確認: タブ切替でアクティブインジケータが移動する
