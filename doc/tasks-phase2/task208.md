# Task 208: モーダルの改善

## フェーズ

Phase 2 - UIデザインブラッシュアップ

## 依存タスク

* Task 201
* Task 209（テーブル共通スタイル）

## 内容

PlayerDetailsModal のビジュアルとアニメーションを改善する。

### やること

* `PlayerDetailsModal.svelte`:
  * 開閉アニメーションを追加:
    * オーバーレイ: opacity 0→1 のフェード（0.2s）
    * モーダル本体: opacity + transform: scale(0.95)→scale(1)（0.2s ease-out）
    * Svelte の `transition:fade` / `transition:scale` またはCSS アニメーション
  * テーブルに交互行背景（zebra stripe）を適用
  * テーブルの行間・パディングを広げる
  * ヘッダーにテーマカラーのアクセント（左ボーダーや背景）
  * ハードコード色をトークンに置換
  * テーブルスタイルを共通クラス（Task 209）に統一
