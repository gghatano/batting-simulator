## 内容

初期ローディング、エラー表示、シミュレーション実行中のフィードバックを改善する。

### やること

* **初期ローディング**（App.svelte）:
  * 「Loading players...」テキスト → スピナー + テキストの組み合わせに変更
  * または スケルトン画面（カード形状のプレースホルダ）
* **エラー表示**（App.svelte）:
  * 赤テキスト → アラートバナーに変更
  * アラートバナー: アイコン（⚠️） + メッセージ + 背景色（danger-50） + 左ボーダー（danger-500）
  * 「再読み込み」ボタンを追加（loadPlayers を再実行）
* **シミュレーション実行中**（SimulationPanel.svelte）:
  * 既存のテキスト表示 → プログレスバー（不確定型：animatedストライプ）に変更
  * ボタンのラベルを「実行中...」に変更 + スピナー表示
* **探索実行中**（LineupSearchPanel.svelte）:
  * 同様にプログレスバーを追加
