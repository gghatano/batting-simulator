# Task 211: 探索結果の表示改善

## 依存タスク

* Task 204

## 内容

LineupSearchPanel の探索結果の表示を改善し、現在打線との差分を可視化する。

### やること

* `LineupSearchPanel.svelte` のリデザイン:
  * **打順の表示改善**:
    * テキスト羅列 → 横並びのバッジ/チップに変更
    * 各チップ: 打順番号 + 選手名を小さなタグとして表示
    * チップのサイズ: font-size 0.75rem, padding 0.15rem 0.4rem, border-radius var(--radius-full)
  * **現在打線との差分ハイライト**:
    * 現在の打線と異なる打順位置のチップを色付き（テーマカラー背景）で表示
    * 同じ位置のチップは灰色背景（変化なし）
  * **「この打順を採用」ボタン**:
    * 各候補行にボタンを追加
    * 押下で lineupStore を更新して打線に反映
  * テーブルのレスポンシブ対応:
    * 横幅が足りない場合は横スクロール（`overflow-x: auto` のラッパー）
  * ハードコード色をトークンに置換
