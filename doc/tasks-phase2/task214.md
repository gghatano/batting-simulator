# Task 214: マイクロインタラクション

## フェーズ

Phase 2 - UIデザインブラッシュアップ

## 依存タスク

* Task 207（打線パネル完成後）
* Task 205（タブ完成後）

## 内容

各種操作にアニメーションフィードバックを追加し、操作感を向上させる。

### やること

* **打線追加時のフィードバック**（LineupSlot.svelte）:
  * 選手がセットされた瞬間にスロットが短くハイライト（pulse アニメーション、0.3s）
  * 背景色が一瞬テーマカラー → 通常色に戻る
* **タブ切替のトランジション**（App.svelte）:
  * タブコンテンツのフェードイン（opacity 0→1、0.2s）
  * Svelte の `transition:fade` または CSS animation
* **ボタンの押下フィードバック**:
  * 全ボタンに `active` 時の `transform: scale(0.97)` を追加
  * `src/styles/global.css` に追加してグローバル適用
* **打線完成時のフィードバック**（LineupPanel.svelte）:
  * 9枠すべて埋まった瞬間にパネル全体に短い緑色のフラッシュ
  * または「シミュレーション開始」ボタンが強調される
* **`prefers-reduced-motion`** 対応:
  * `@media (prefers-reduced-motion: reduce)` で全アニメーションを無効化

## 完了条件

- [ ] 打線に選手追加時、該当スロットがハイライトする
- [ ] タブ切替時にコンテンツがフェードインする
- [ ] ボタン押下時に微小な縮小エフェクトがある
- [ ] `prefers-reduced-motion` でアニメーションが無効化される
- [ ] `npm run build` が成功する

* 手動確認: タブ切替 → コンテンツがスムーズに表示される
* 手動確認: ボタンクリック → 微小な反応がある
