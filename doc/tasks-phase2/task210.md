# Task 210: SimulationSummary の視認性向上

## フェーズ

Phase 2 - UIデザインブラッシュアップ

## 依存タスク

* Task 201
* Task 204
* Task 209

## 内容

SimulationSummary の主要指標を目立たせ、得点分布を視覚化する。

### やること

* `SimulationSummary.svelte` のリデザイン:
  * **主要指標のカード化**:
    * 平均得点を大きなフォント（2rem+）で表示
    * 中央値を同様に目立たせる
    * 2つを横並びのカード/バッジとして配置
  * **サブ指標**:
    * P10/P90、最大/最小を控えめに表示（主要指標の下に小さく）
  * **得点分布のバーチャート化**:
    * CSS のみで水平バーチャートを実装
    * 各得点の行: ラベル + バー（widthを出現率に比例）+ パーセンテージ
    * バーの色にテーマカラーを使用
    * `<details>` による折りたたみは維持しても可（デフォルトで開いてもよい）
  * ハードコード色をトークンに置換

## 完了条件

- [ ] 平均得点・中央値が大きなフォントで目立つ
- [ ] P10/P90 が控えめに表示される
- [ ] 得点分布がバーチャート（CSSのみ）で表示される
- [ ] ハードコード色がトークンに置換されている
- [ ] `npm run build` が成功する
